# 知识图谱问答系统

## 项目结构

```
kg-qa-system/
├── backend/                 # FastAPI 后端
│   ├── api/                # API 路由
│   ├── core/               # 核心配置
│   ├── schemas/            # Pydantic 模型
│   ├── services/          # 业务服务
│   │   ├── deepseek_service.py    # DeepSeek API
│   │   ├── neo4j_service.py       # Neo4j 图数据库
│   │   └── document_parser.py     # 文档解析
│   └── requirements.txt
├── frontend/               # Vue3 前端
│   ├── src/
│   │   ├── api/          # API 调用
│   │   ├── views/        # 页面组件
│   │   └── router/       # 路由配置
│   └── package.json
└── documents/             # 课程文档目录
```

## 环境准备

### 1. 安装后端依赖

```bash
cd backend
pip install -r requirements.txt
```

### 2. 配置环境变量

复制 `.env.example` 为 `.env`，填入你的配置：

```bash
cp .env.example .env
```

编辑 `.env`:
- `DEEPSEEK_API_KEY`: 你的 DeepSeek API Key
- `NEO4J_PASSWORD`: Neo4j 数据库密码

### 3. 安装 Neo4j

下载并启动 Neo4j (社区版):
- https://neo4j.com/download/

启动后访问 http://localhost:7474

### 4. 安装前端依赖

```bash
cd frontend
npm install
```

## 启动服务

### 启动后端

```bash
cd backend
uvicorn api.main:app --reload --host 0.0.0.0 --port 8000
```

### 启动前端

```bash
cd frontend
npm run dev
```

访问 http://localhost:3000

## 使用流程

1. **上传文档**: 在"文档管理"页面上传 PDF/Word/TXT 文件
2. **抽取三元组**: 点击"抽取三元组"从文档中提取知识
3. **查看图谱**: 在"知识图谱"页面可视化查看实体关系
4. **智能问答**: 在"智能问答"页面提问，系统优先从知识图谱检索回答
