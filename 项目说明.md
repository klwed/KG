# 知识图谱问答系统

## 项目简介

这是一个基于知识图谱的智能问答系统，主要用于教育领域，针对"离散数学"和"信息安全"等计算机课程。该系统实现了从教学文档自动构建知识图谱，并提供智能问答功能。

## 功能特性

### 1. 知识图谱构建
- 支持 PDF、DOCX、TXT 格式的教学文档解析
- 基于 LangChain + DeepSeek LLM 自动抽取知识三元组
- 将三元组存储到 Neo4j 图数据库

### 2. 智能问答
- 支持仅基于知识图谱或结合 LLM 混合问答
- 可视化展示全局/子图谱
- 支持多轮对话上下文
- 点击节点查看详情

### 3. 学生管理
- 导入学生成绩 CSV 数据
- 自动分析学生薄弱点与强项
- 将学生数据转换为三元组导入图谱
- 生成班级掌握情况报告

### 4. 个人学习查询
- 学生可查看自己的知识点掌握情况
- 问答时可结合个人成绩给出针对性回答

### 5. 用户权限系统
- 教师：管理文档、构建图谱、生成邀请码、管理学生
- 学生：查看个人成绩、进行问答

## 知识图谱结构

```
课程 → 章节 → 知识点 → 知识标签
              ↓
         计算思维维度（分解、模式识别、抽象、算法思维、评估）

学生 --[掌握]--> 知识点 --[体现]--> 计算思维
```

## 快速开始

### 环境要求
- Python 3.8+
- Node.js 16+
- Neo4j 4.x+

### 后端安装

```bash
cd backend
pip install -r requirements.txt
```

配置 `.env` 文件：

```
DEEPSEEK_API_KEY=your_api_key
NEO4J_PASSWORD=your_password
```

启动后端：

```bash
python -m uvicorn api.main:app --reload
```

### 前端安装

```bash
cd frontend
npm install
npm run dev
```

### 访问系统

打开浏览器访问 http://localhost:5173

## API 文档

启动后端后访问 http://localhost:8000/docs 查看 API 文档。
