# 知识图谱智能问答系统

基于混合检索增强生成（RAG）的智能问答系统，支持知识图谱构建、向量检索、学生成绩分析等功能。

## 项目简介

本系统是一个教育领域的知识图谱智能问答平台，主要功能包括：

- **知识图谱构建**：从文档中自动抽取知识三元组，存储到 Neo4j 图数据库
- **混合检索问答**：结合向量检索（FAISS）和图谱检索（Neo4j）的 RAG 问答
- **学生管理**：学生成绩分析、知识点掌握情况追踪
- **计算思维关联**：将知识点与计算思维维度关联分析

## 快速开始

### 环境要求

- Python 3.10+
- Node.js 18+
- Neo4j 5.x

### 1. 安装后端依赖

```bash
cd backend
pip install -r requirements.txt
```

### 2. 配置环境变量

```bash
cp .env.example .env
```

编辑 `.env`:
```env
DEEPSEEK_API_KEY=your_api_key
NEO4J_PASSWORD=your_password
```

### 3. 安装前端依赖

```bash
cd frontend
npm install
```

### 4. 初始化数据库

```bash
cd backend
python -c "from core.database import init_db; init_db()"
```

### 5. 启动服务

```bash
# 终端1 - 后端
cd backend
uvicorn api.main:app --reload --host 0.0.0.0 --port 8000

# 终端2 - 前端
cd frontend
npm run dev
```

访问 http://localhost:3000

## 使用流程

1. **注册登录**：使用邀请码注册（首个注册用户为教师）
2. **上传文档**：上传课程文档（PDF/Word/TXT）
3. **向量化**：将文档向量化存入 FAISS
4. **抽取三元组**：从文档中提取知识
5. **导入图谱**：存入 Neo4j
6. **智能问答**：输入问题，系统检索后生成答案

## API 接口

### 认证 `/api/auth`
| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /register | 用户注册 |

| 方法 | 路径 | 说明 |

### 问答 `/api/qa`
| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /ask | 问答接口 |

### 学生 `/api/students`
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /list | 学生列表 |
| GET | /overview | 掌握情况汇总 |

## 问答流程

```
用户提问
    ↓
┌─────────────────────────────────┐
│  双路检索                        │
│  ├─ FAISS 向量检索              │
│  └─ Neo4j 图谱检索              │
└─────────────────────────────────┘
    ↓
结果融合 → 构建上下文
    ↓
DeepSeek LLM 生成答案
    ↓
返回：答案 + 相关三元组 + 子图
```
